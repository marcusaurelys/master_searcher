\begin{enumerate}
  \item \textbf{Comparison leader:} Bird-BM is consistently best or near-best in comparison count.
  \item \textbf{Runtime leader:} Greedy BM is the most reliable wall-clock winner on tested practical scenarios.
  \item \textbf{Metric divergence is real:} implementation overhead changes winner selection.
  \item \textbf{Decision rule:} pick algorithm by workload and objective, not by one global metric.
\end{enumerate}
\keymsg{For this codebase, \textbf{Greedy BM} is the default deployment pick when runtime is primary.}
\end{frame}
